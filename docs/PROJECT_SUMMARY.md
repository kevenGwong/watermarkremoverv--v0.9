## 🔧 技术架构

### 设计模式
- **工厂模式**: 模型创建和管理
- **策略模式**: 不同处理策略
- **管道模式**: 处理流程编排
- **观察者模式**: 事件通知

### 核心组件
- **BaseModel**: 模型抽象基类
- **WatermarkProcessor**: 主处理器
- **ModelFactory**: 模型工厂
- **ConfigManager**: 配置管理器

### 数据流
```
输入图像 → 预处理 → 检测/分割 → 掩码生成 → 图像修复 → 后处理 → 输出
```
